# Итог выполнения HW01 – базовые операции с `numpy`

## Кратко

В ноутбуке `HW01/HW01.ipynb` выполняются базовые упражнения по `numpy`:
создание массивов, проверка форм/типов и простые агрегаты.

## Структура

```text
homeworks/
  HW01/
    HW01.ipynb
    README.md
```

## Данные

Внешних данных нет. Используются вручную заданные массивы `numpy`.

## Как воспроизвести

### 1) Зависимости

Минимально нужна библиотека:

* `numpy`

Установка:

```bash
pip install numpy
```

Если используешь `uv`, то `pyproject.toml` лежит в корне репозитория,
поэтому зависимости ставятся так:

```bash
cd <корень-репозитория>
uv sync
```

### 2) Запуск

Открой ноутбук `homeworks/HW01/HW01.ipynb` и выполни все ячейки.

Пример через `uv` (при наличии Jupyter):

```bash
cd <корень-репозитория>
uv run jupyter lab
```

## Методология

* создание массивов через `np.array`, `np.zeros`, `np.ones`, `np.arange`
* проверка формы и типа (`shape`, `dtype`)
* вычисление простых агрегатов (`sum`, `mean`, `max`)

## Результаты

* продемонстрированы базовые операции над массивами и вывод агрегатов
