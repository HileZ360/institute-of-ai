# Итог выполнения HW02 – первичный EDA на табличных данных

## Кратко

В ноутбуке `HW02/HW02.ipynb` выполняется первичный анализ небольшого табличного
датасета `S02-hw-dataset.csv`: описательная статистика, проверка пропусков и
простые визуализации.

## Структура

```text
homeworks/
  HW02/
    HW02.ipynb
    README.md
    S02-hw-dataset.csv
    figures/
      hist_age.png
```

## Данные

Файл: `S02-hw-dataset.csv`

Колонки:

* `user_id` – идентификатор
* `age` – возраст
* `country` – страна
* `purchases` – число покупок
* `revenue` – суммарная выручка

Сводка по датасету:

* 41 строка, 5 колонок
* 2 пропуска
* типы: `age` – float, `country` – категория (object), остальные – числовые

## Как воспроизвести

### 1) Зависимости

Нужны библиотеки (минимально):

* `pandas`, `numpy`, `matplotlib`

Установка (пример через pip):

```bash
pip install pandas numpy matplotlib
```

Если используешь `uv`, то `pyproject.toml` лежит в корне репозитория,
поэтому зависимости ставятся так:

```bash
cd <корень-репозитория>
uv sync
```

### 2) Запуск

Открой ноутбук `homeworks/HW02/HW02.ipynb` и выполни все ячейки.

Пример через `uv` (при наличии Jupyter):

```bash
cd <корень-репозитория>
uv run jupyter lab
```

## Методология

* загрузка CSV через `pandas`
* проверка пропусков и базовые статистики (`describe`)
* группировки по `country`, анализ распределений
* визуализация: гистограмма возраста и boxplot

## Результаты

* построена гистограмма возраста (`homeworks/HW02/figures/hist_age.png`)
* получены базовые статистики и распределения по странам
